# 📌 Project Overview

This project demonstrates the complete data analytics lifecycle, from raw data processing to business intelligence dashboard creation.

Using a retail customer shopping dataset, the project explores purchasing patterns, customer demographics, seasonal trends, and product preferences to help businesses make data-driven decisions.

The project covers:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Business Analysis
- Interactive Power BI Dashboard
- Business Recommendations

---

# 🎯 Business Objective

The objective of this project is to answer important business questions such as:

- Who are the most valuable customers?
- Which products generate the highest sales?
- Which customer segments contribute the most revenue?
- How do age and gender affect purchasing behavior?
- Which seasons have the highest shopping activity?
- Which payment methods are preferred by customers?
- What recommendations can increase customer retention and revenue?
